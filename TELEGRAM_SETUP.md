# Рассылка в Telegram

## 1. Деплой на Vercel

1. Зайди на [vercel.com](https://vercel.com), подключи репозиторий wLife
2. В настройках проекта → Environment Variables добавь:
   - `BOT_TOKEN` = токен бота (из @BotFather)
3. Деплой — получишь URL вида `https://wlife-xxx.vercel.app`

## 2. Настройка в приложении

1. В приложении: **Настройки** → **Telegram для рассылки**
2. Введи URL API: `https://твой-проект.vercel.app`
3. Нажми **+ Добавить** — введи имя (как в расписании), затем укажи @username в Telegram

## 3. Важно

- **Токен бота** хранится только на сервере (Vercel), в приложении его нет
- Если токен утёк — отзови его в @BotFather и создай новый
